## How to export each one (web size)
Think of a print file as a giant cinema poster, and a web file as a phone
wallpaper. Same picture — the web one is just lighter so the page loads fast.

In Photoshop / Illustrator:
  File > Export > Export As…
  - Format: JPG
  - Width: about 1600 px on the LONGEST side (height auto)
  - Quality: ~70%
  - Aim for under ~400 KB per file
Then rename to the "Save it as" name and drop it in this folder.

## Tip
The Tech21 / DECODED / Cricut / Vephos sources are already JPGs around
150–600 KB, so you can often just rename + save with almost no shrinking.
Only the Hong Yu packaging and the US booth banner are big print files that
really need the export step above.
